---

## ⚡ PASSO 1: CRIAR CONTA NO NEON

1. **Acesse:** https://neon.tech
2. **Clique em:** "Sign Up" ou "Get Started"
3. **Faça login com:**
   - GitHub (recomendado)
   - Google
   - Email

---

## ⚡ PASSO 2: CRIAR PROJETO

1. **Após login**, clique em **"Create Project"**
2. **Configure:**
   - **Project Name:** `evolution-api-barbearia` (ou nome de sua escolha)
   - **Region:** Escolha a mais próxima (ex: `São Paulo` ou `US East`)
   - **PostgreSQL Version:** `16` (ou a mais recente)
3. **Clique em:** "Create Project"
4. **Aguarde 30-60 segundos** para criar

---

## ⚡ PASSO 3: OBTER CONNECTION STRING

**Após criar o projeto:**

1. **Na página do projeto**, você verá a **Connection String** diretamente
2. **Formato:**
   ```
   postgresql://usuario:senha@ep-xxx-xxx.region.aws.neon.tech/neondb?sslmode=require
   ```
3. **Copie a connection string COMPLETA**

**💡 Dica:** A connection string já está pronta para usar!

---

## ⚡ PASSO 4: CONFIGURAR EVOLUTION API

**Execute:**
```powershell
.\configurar-com-neon.ps1
```

**Cole a connection string quando solicitado.**
