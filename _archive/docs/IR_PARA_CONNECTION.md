# ⚡ IR PARA A PÁGINA CORRETA

## 🎯 VOCÊ ESTÁ EM "SETTINGS" - PRECISA IR PARA "CONNECTION"

### O QUE FAZER AGORA:

1. **No menu lateral esquerdo**, procure por:
   ```
   DATABASE MANAGEMENT
     Connection  ← CLIQUE AQUI
     Settings    ← (você está aqui)
   ```

2. **Clique em "Connection"** (não "Settings")

3. **Na página "Connection"**, você verá a connection string completa!
